# ISO 27001:2022 GRC Toolkit

> **Note:** ABC India Pvt. Ltd. is a fictional organisation 
> created for portfolio and learning purposes only.
> All findings and documentation are simulated.

## Overview
Complete Information Security Management System (ISMS) 
documentation built for a mock fintech organisation — 
ABC India Pvt. Ltd. Covers full ISO 27001:2022 implementation 
lifecycle from scoping through gap assessment, risk treatment, 
and policy development.

## Organisation Profile (Mock)
- **Name:** ABC India Pvt. Ltd.
- **Industry:** Fintech / Payment Processing
- **Size:** 100 employees
- **Locations:** Bengaluru, Mumbai
- **Compliance Requirements:** ISO 27001:2022, RBI Guidelines, PDPB

## Repository Structure
| Folder | Contents |
|--------|----------|
| 01-scope-and-context | ISMS scope, context, interested parties |
| 02-risk-assessment | Asset inventory, threat matrix, risk register |
| 03-gap-assessment | Current vs required controls — all 93 Annex A |
| 04-statement-of-applicability | SoA with justifications |
| 05-policies | Information security policy suite |
| 06-multiframework-mapping | ISO 27001 vs NIST CSF vs SOC 2 |

## Key Deliverables
- Risk Register with 20+ identified risks
- Gap Assessment covering all 93 ISO 27001:2022 Annex A controls
- Statement of Applicability with justifications
- 4 core security policies
- Multiframework control mapping

## Tools and Frameworks
